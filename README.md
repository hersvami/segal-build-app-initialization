# Segal Build

Professional construction quoting and variation management for Australian domestic builders.

**Live:** https://segal-build-app.web.app

---

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
firebase deploy --only hosting
```
